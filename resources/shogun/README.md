# Shogun

## 关键词
`机器学习`, `C++`, `SWIG`

## 基本介绍
Shogun是用C++实现的机器学习库，类似于sklearn, 利用SWIG暴露接口给Python, Octave, Java / Scala, Ruby, C#, R, Lua.
