# Brl-cad

## 关键词
`C`,`C++`, `CAD`, `machine learning` 

## 基本介绍

[Brl-cad](https://brlcad.org/) 已经申请很多年了，每年大概会有10-12个人入选，虽然同属于CAD大家族，但比热门的FreeCad和CGAL简单很多。

## 申请指南

brl-cad申请是必须要提至少一个pr的，并且需要写在自己的research proposal中。然后最好多在zulip进行交流，并且介绍自己，然后公布自己的research proposal，多刷存在感对申请被接受有很大帮助。
